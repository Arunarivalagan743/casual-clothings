STYLE REQUIREMENTS:

6. ADMIN WORKFLOW:

- Is user authenticated? (Yes/No)
- Is product in stock? (Yes/No)

- JWT Token Validation
- MongoDB Database Operations
- Razorpay Payment Gateway
- Email Notifications
- Stock Validation
- Size-based Pricing

LAYOUT SUGGESTION:
- Horizontal flow from left to right
- Multiple parallel processes where applicable
- Clear separation between user flow and admin flow
